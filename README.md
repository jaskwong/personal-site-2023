# personal-site-2023

A personal porfolio to display projects I've worked on.

Typescript, CSS, Vue.js, Yarn

## Run Locally
1. run `yarn dev`

## Build and Deploy
1. build with 'yarn build'
2. add, commit, and push changes to git
3. deploy with 'yarn deploy'
